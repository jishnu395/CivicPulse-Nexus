# CivicPulse Nexus

A microservices-based civic grievance management system built using Spring Boot and Spring Cloud.

## Tech Stack

- Java
- Spring Boot
- Spring Cloud Gateway
- Netflix Eureka
- PostgreSQL
- Maven

## Project Structure

```text
CivicPulse-Nexus/
├── discovery-service
├── gateway-service
├── user-service
├── citizen-service (Coming Soon)
├── grievance-service (Coming Soon)
```
